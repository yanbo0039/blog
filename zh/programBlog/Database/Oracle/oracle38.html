                                                                                                                                # 38-PlSQL Developer如何设置备份文件禁止不再生成~sql文件.mdopen in new window

                                                                                                                                # 故事背景

                                                                                                                                如图所示，通过plsql保存的.sql文件都会自动生成一个.~sql文件

                                                                                                                                这个文件的作用是用于备份，但是我们不想要这.~sql文件怎么办呢？

                                                                                                                                # 解决办法

                                                                                                                                Confbd268igure --> preferences -->Files --> Backup -->右侧第一个勾选disable

                                                                                                                                # 测试结果

                                                                                                                                1. 删除掉所有的.~sql文件

                                                                                                                                2. open --->sqlScript--> 选择所有的sql文件打开后保存而后在对应的文件目录中查看是否有.~sql文件存在

                                                                                                                                3.如图所示并没有出现对应的.~sql文件则表示修改成功

                                                                                                                                以上就是我关于 PlSQL Developer如何设置备份文件禁止不再生成~sql文件 知识点的整理与总结的全部内容，希望对你有帮助。。。。。。。
