                                                                                        # BUG回顾

                                                                                        • 最近启动一个JAVAEE项目时报错如下,多次尝试解决找不到原因，于是只能暴力解决了。

Multiple Contexts have a path of "/tickets".

                                                                                        # 解决方法

                                                                                        • 删除Servers文件夹，以及控制台的Servers里的Tomcat，而后再次添加即可了
